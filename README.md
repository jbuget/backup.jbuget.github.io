## jbuget.github.io

Démarrer le serveur de développement :

```
$ bundle exec jekyll serve
```

ou tout simplement : 

```
$ jekyll s
```


Accéder à l'application en local :

```
http://127.0.0.1:4000
```

Démarrer le serveur de développement avec affichage des brouillons :


```
$ jekyll server --drafts
```
